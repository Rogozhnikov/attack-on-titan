# Attack on Titan: Recruit

Разработка мини-игры по мотивам одноименной манги.

Игрок создает свою учетку, выбирает режим подготовки и обложку персонажа из ростера, а затем попадает на домашнюю страницу столицы внутри стены Сина. В городе доступны ячейки строительства, накопление ресурсов и первая вылазка за стену.

Опубликованная версия: https://titan.wastelandking.ru

## Состав проекта

- `apps/web` - React + TypeScript + Vite фронтенд.
- `apps/api` - Fastify + TypeScript API.
- PostgreSQL на проде хранит игроков и состояние города.

## Ресурсы

- Еда: базовая добыча через поля каждые 30 секунд.
- Дерево: добыча через лесной двор каждые 45 секунд.
- Камень: добыча через каменоломню каждые 50 секунд.
- Железо: добыча через мастерскую каждые 70 секунд.
- Люди: прирост через приют каждые 120 секунд или быстрый прирост через вылазки.

## Локальный запуск

```bash
npm install
npm run build
```

Для API можно задать `DATABASE_URL`. Без него API использует JSON-хранилище для локальной разработки.
