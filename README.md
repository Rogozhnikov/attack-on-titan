# Attack on Titan: Recruit

Разработка мини-игры по мотивам одноименной манги.

Игрок создает свою учетку, выбирает режим подготовки и обложку персонажа из ростера. Персонажи доступны всем игрокам, а уже созданную чужую учетку занять нельзя.

Опубликованная версия: https://titan.wastelandking.ru

## Состав проекта

- `index.html` - стартовый экран игры.
- `server.js` - простой API для хранения игроков.
